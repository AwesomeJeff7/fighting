Java Fighting Game

About the Game

A fast-paced 2D fighting game created in Java. Challenge your friend in an intense one-on-one battle! Features include health bars, blocking mechanics, and real-time combat.

Game Controls

Player 1 Controls:

  • A - Move Left
  • D - Move Right
  • F - Attack
  • G - Block (hold)

Player 2 Controls:

  • Left Arrow - Move Left
  • Right Arrow - Move Right
  • K - Attack
  • L - Block (hold)

How to Run the Game

  1. Make sure you have Java installed on your computer
  2. Download all game files (Fighter.java, GamePanel.java, and FightingGame.java)
  3. Open terminal/command prompt in the game directory
  4. Compile the game: javac *.java
  5. Run the game: java FightingGame

Game Features

  • Two-player combat system
  • Real-time health tracking
  • Attack and block mechanics
  • Collision detection
  • Victory conditions
